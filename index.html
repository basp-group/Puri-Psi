Introduction

Puri-Psi is an open-source code that provides functionality to address radio interferometric imaging problems using state-of-the-art convex optimisation algorithms.

Puri-Psi is currently available as a collection of independent Matlab toolboxes. A C++ library with partial Python support is currently under development and will be provided in due course. For further details on the methods currently implemented, please see the following references.

Puri-Psi is made available under the license described below.

Matlab

Puri-Psi is currently composed of several independent Matlab toolboxes devoted to specific radio radio-interferometric imaging problems. Further details on each of these methods can be directly found on their respective Github repository, which can be separately accessed by clicking on the following links.

  • Scalable splitting algorithms for big data interferometric imaging
  • Accelerated splitting algorithm for radio-interferometric imaging
  • Joint imaging and DDE calibration
  • Fourier dimensionality reduction for big data interferometric imaging
  • Adaptive preconditioned primal-dual for radio-interferometry
  • Joint Stokes imaging in radio-interferometry

C++ / Python

A C++ implementation of the Puri-Psi library is currently under development, and will be made available on this page in due course.
