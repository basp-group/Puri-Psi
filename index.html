Introduction

Puri-Psi is an open-source code developed within the BASP group (Biomedical and Astronomical Signal Processing) headed by Prof. Wiaux that provides functionality to address radio interferometric imaging problems using state-of-the-art optimisation algorithms and deep learning.

The name Puri-Psi builds from former software Purify by Prof. Wiaux and collaborators, with the extension -Psi standing for "Parallel Proximal Scalable Imaging".

Puri-Psi is currently available (under the license described below) as a collection of independent MATLAB toolboxes, as well as C++ codes, all described below and available on Github.

Matlab

A collection of independent Matlab toolboxes is available below, covering the following functionalities: scalable monochromatic imaging (SARA and AIRI), scalable wide-band imaging (HyperSARA and Faceted HyperSARA), scalable Polarised imaging (Polarized SARA), joint calibration and imaging (Polca SARA), and scalable uncertainty quantification (BUQO).

SARA - Scalable monochromatic imaging

  • Scalable splitting algorithms for big data interferometric imaging
  • Accelerated splitting algorithm for radio-interferometric imaging
  • Fourier dimensionality reduction for big data interferometric imaging
  • Adaptive preconditioned primal-dual for radio-interferometry

HyperSARA - Scalable wide-band imaging

  • Wideband Super-resolution Imaging in Radio Interferometry via Low Rankness and Joint Average Sparsity Models (HyperSARA)
  • Parallel faceted imaging in radio interferometry via proximal splitting (Faceted HyperSARA)

Polarized SARA - Scalable polarised imaging

  • Sparse interferometric Stokes imaging under polarization constraint (Polarized SARA)

Joint calibration and imaging

  • Non-convex optimization for self-calibration of direction-dependent effects in radio-interferometric imaging
  • Time-Regularized Blind Deconvolution Approach for Radio Interferometry
  • Self direction-dependent calibration for full polarization radio-interferometric sparse imaging (Polca SARA)
  • Monochromatic joint DDE calibration and imaging applied to Cygnus A observations from the VLA.

BUQO - Scalable uncertainty quantification

  • Scalable Bayesian uncertainty quantification in imaging inverse problems via convex optimization

AIRI - AI regularisation for radio interferometry

C++

A C++ code for both scalable monochromatic imaging (SARA) and scalable wide-band imaging (HyperSARA) is available here.
